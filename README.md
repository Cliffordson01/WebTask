# 🚀 Exercícios de HTML e CSS

Este repositório reúne uma coleção de **15 exercícios práticos** com foco no domínio de **HTML5** e **CSS3**.  
Cada exercício propõe o uso obrigatório de uma **tag HTML** e uma **propriedade CSS**, estimulando a aplicação semântica e estética em páginas web.

## 1. Página de Introdução
- **Tag HTML**: `<h1>`
- **Propriedade CSS**: `background-image`
- **Descrição**:  
  O título principal `<h1>` apresenta o projeto **Aether**, com destaque visual reforçado por uma imagem de fundo aplicada via `background-image` na classe `.background`.

- **Propriedade CSS**: `border-radius`
- **Descrição**:  
  A imagem de perfil `<img>` recebe um tratamento visual com `border-radius: 50%`, resultando em um avatar perfeitamente circular.

## 3. Página de Portfólio
- **Tag HTML**: `<div>`
- **Propriedade CSS**: `display: grid`
- **Descrição**:  
  Cada item do portfólio é estruturado em `<div>`, organizado visualmente em uma malha responsiva através do uso de `display: grid` na classe `.portfolio-grid`.
